# Firebase konfiguracija (skica)

- Auth: Google prijava
- Storage: privremeni upload slike šalice (auto-delete nakon obrade)
- Firestore: evidencija čitanja (uz user consent)
- Remote Config: ključevi za feature flagove i verzioniranje modela

## Remote Config ključevi (primjeri)
- rc_min_app_version
- rc_free_reads_per_month
- rc_premium_enabled
- rc_model_version
